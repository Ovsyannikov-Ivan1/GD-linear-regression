# Градиентный спуск для линейной регрессии

В этом проекте реализованы несколько модификаций градиентного спуска для обучения линейной регрессии с нуля. Работа включает как реализацию оптимизаторов и функций потерь, так и их экспериментальное сравнение на реальных данных.

## О проекте

Цель проекта — изучить, как различные методы градиентной оптимизации работают в задаче линейной регрессии, и сравнить их по качеству и скорости сходимости.

В рамках проекта были:
- выведены и реализованы формулы для линейной регрессии;
- реализованы аналитическое решение и итерационные методы оптимизации;
- исследованы различные стратегии выбора шага обучения;
- проведено сравнение методов на реальном датасете;
- изучено влияние размера батча, регуляризации и альтернативных функций потерь.

## Реализованные методы

В проекте реализованы следующие оптимизаторы:

- Vanilla Gradient Descent
- Stochastic Gradient Descent (SGD)
- Stochastic Average Gradient (SAG)
- Momentum
- Adam

Также были реализованы:
- аналитическое решение задачи линейной регрессии;
- решение через SVD для случая вырожденной матрицы признаков;
- learning rate schedules;
- L2-регуляризация;
- альтернативные функции потерь: LogCosh и Huber loss.

## Структура работы

Проект включает несколько основных частей:

### 1. Линейная регрессия
- вывод градиента MSE в матричном виде;
- вывод аналитического решения;
- реализация собственной линейной регрессии;
- реализация аналитического решения через обращение матрицы и через SVD.

### 2. Реализация градиентного спуска
- реализация базового цикла оптимизации;
- реализация нескольких вариантов градиентного спуска;
- поддержка различных learning rate schedules.

### 3. Тестирование
- проверка корректности реализованных классов на синтетических данных;
- сравнение собственной реализации с sklearn.

### 4. Работа с данными
- анализ датасета объявлений о продаже автомобилей;
- обработка выбросов;
- исследование признаков;
- разделение признаков на категориальные, числовые и прочие;
- подготовка данных с помощью OneHotEncoder и StandardScaler;
- разбиение на train / validation / test.

### 5. Сравнение методов оптимизации
- подбор оптимального learning rate;
- сравнение методов по ошибке, качеству и числу итераций;
- построение графиков сходимости.

### 6. Исследование размера батча
- серия запусков SGD с разными batch size;
- сравнение времени сходимости;
- анализ числа итераций до остановки.

### 7. Регуляризация
- реализация L2-регуляризации;
- подбор параметров регуляризации;
- сравнение качества моделей с регуляризацией и без неё.

### 8. Альтернативные функции потерь
- вывод и реализация LogCosh loss;
- вывод и реализация Huber loss;
- сравнение результатов с MSE.

## Датасет

В экспериментальной части используется датасет объявлений о продаже автомобилей с немецкого eBay.
